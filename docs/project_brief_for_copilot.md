# Proje Özeti

## Proje adı
Dijitalleştirilmiş Otosanayi

## Amaç
Otosanayi işletmeleri ile müşteriler arasında dijital köprü kuran web tabanlı uygulama geliştirmek.

## Temel roller
- Müşteri
- İşletme sahibi
- Yönetici

## Minimum özellikler
- İşletme listeleme
- İşletme detay görüntüleme
- Yorum ve puan sistemi
- Ortalama fiyat bilgisi
- İş emri oluşturma
- İş emri durum takibi
- Register / login / JWT
- Rol bazlı erişim

## Teknoloji
- Backend: Python + FastAPI
- Frontend: React
- Veritabanı: PostgreSQL
- Auth: JWT

## Mimari yaklaşım
- Frontend, backend ve veritabanı katmanlı yapı ile ayrılır
- Kod sade ve teslim odaklı tutulur
- Gereksiz özellik eklenmez

## Çalışma yaklaşımı
- Küçük adımlarla ilerle
- Önce plan ver, sonra kod üret
- Koddan önce değişecek dosyaları söyle
- Minimum değişiklik yap
- Çalışan çıktıyı doğrulat
- Commit mesajı öner
- Her commit’ten sonra push öner

## Mevcut durum
- Repo oluşturuldu
- Backend iskeleti kuruldu
- Frontend iskeleti kuruldu
- PostgreSQL bağlantısı test edildi
