# Copilot için çalışma talimatları

Bu repo bir bilgisayar mühendisliği bitirme projesidir.
Amaç: sade, çalışan, teslim edilebilir bir web uygulaması üretmek.

## Teknik sınırlar
- Backend: FastAPI
- Frontend: React
- DB: PostgreSQL
- Auth: JWT

## Davranış kuralları
- Önce kısa plan ver
- Koddan önce değişecek dosyaları listele
- Aynı anda çok fazla dosya değiştirme
- Backend ve frontend işlerini mümkünse ayrı ele al
- Mevcut yapıyı koru
- Gereksiz kütüphane ekleme
- Küçük ve açıklanabilir değişiklik yap
- Test veya doğrulama adımı ekle
- Gerekirse commit mesajı öner
