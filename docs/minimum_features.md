# Minimum özellikler

## Minimum teslim özellikleri
- Kullanıcı kayıt olabilmeli ve giriş yapabilmeli.
- Kullanıcı rolü müşteri / işletme sahibi / yönetici olarak ayrılmalı.
- Müşteri işletmeleri listeleyebilmeli.
- Müşteri işletme detayını görebilmeli.
- Müşteri yorumları ve puanları görebilmeli.
- İşletme sahibi iş emri açabilmeli.
- İşletme sahibi iş emri durumunu güncelleyebilmeli.
- Müşteri kendi iş emrinin durumunu görebilmeli.
- Sistem hizmet türüne göre ortalama fiyat gösterebilmeli.
- Yetkisiz kullanıcı korumalı alanlara erişememeli.

## Nice-to-have ama şu an minimum değil:
- Harita/konum entegrasyonu
- Admin moderasyon ekranı
- Gelişmiş filtreleme
- Dashboard grafikleri
- Dosya/fotoğraf yükleme
- Bildirim sistemi
