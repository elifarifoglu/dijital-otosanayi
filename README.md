# Dijitalleştirilmiş Otosanayi

Otosanayi işletmeleri ve müşterileri için web tabanlı uygulama.

## Proje Özeti
Bu proje; müşterilerin işletmeleri inceleyebilmesini, yorum ve puanlarını görebilmesini, belirli hizmetler için ortalama fiyat bilgisi alabilmesini ve araç işlem sürecini takip edebilmesini amaçlar. İşletme sahipleri ise işletme bilgilerini, iş süreçlerini ve araç durumlarını yönetebilir.

## Kullanılan Teknolojiler
- Backend: Python, FastAPI
- Frontend: React
- Veritabanı: PostgreSQL
- Authentication: JWT

## Klasör Yapısı
- `backend/`: sunucu tarafı
- `frontend/`: kullanıcı arayüzü
- `docs/`: proje belgeleri
